<h1>Disassemble & Rebuild a Desktop PC</h1>


<h2>Description</h2>
<p>This project showcases my ability to disassemble, inspect, and reassemble a desktop PC to enhance my technical skills and hands-on experience with computer hardware.</p>

<p>In this project I will explain each step of the process in detail, as well as any computer components that I remove or install.</p>

<br />

<h2>Environments Used </h2>

- <b>Oracle VirtualBox</b>
- <b>Windows Server 2016</b>
- <b>Windows 10</b> 


<h2>Tools Used</h2>

- <b>Active Directory Users and Computers</b> 
- <b>Server Manager</b>
- <b>Command Prompt</b> 
- <b>Control Panel</b>
- <b>Remote Desktop Connection</b>
- <b>File Explorer</b>
- <b>Group Policy</b>
